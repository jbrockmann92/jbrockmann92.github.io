Stock Score

Technologies: C#, Google Search API, Alpha Vantage API, html, CSS

Stock Score is a .Net Core web application for financial advisors and amateur investors.

Problem: this application fulfills the need of the average consumer not having an accessible and simple resource for deciding on an investment.

The user simply registers for a profile and searches for a stock using the stock's symbol. They choose a time frame they would like to invest for, whether day, week, or month. The application searches Google headlines over the past day, week, or month depending on search parameters, and the stock using the Alpha Vantage API, and calculates whether the stock is a good investment for the given time frame.

FlatMatch

Technologies: C#, JavaScript, Google Maps API, html, CSS

FlatMatch is a roommate matching app for people who have an extra room.

Problem: This application addresses the need for many people to find roommates with whom they can get along well.

A user can register either as a leaseholder (apartment or house resident) or renter (person looking for a place). Both types of user choose from a list of nine preferences on a scale of 1-5, and the application matches leaseholders and renters together in a given area based on how compatible they are. Users also have a simple profile page with a picture and 'About Me' section.

Trash Collection

Technologies: C#, Google Maps API, JavaScript, html, CSS

Trash Collector is a .Net Core web application for use by a municipal trash collection department

Problem: This application is a solution to the needs of many municipalities for a simple, effective tool for their residents and employees.

The user can be registered as either an employee or customer. Customers can choose their preferred pickup day, set their address, suspend and resume their pickup service, and choose a one-time pickup for a specific day and time. Employees can see a list of their pickups for the day and the addresses, as well as a map with a pin to show the location.
